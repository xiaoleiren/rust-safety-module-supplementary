# Teaching Materials — Rust Safety Module

This directory contains the complete teaching materials for the Rust safety module described in the paper:

**"Compiler-Enforced Safety as a Learning Object: Teaching Memory Safety and Concurrency with Rust"**

## 📥 Access During Blind Review

This anonymized repository includes student-facing lab handouts, rubric templates, annotation worksheets, anonymized data summaries, and supplementary analyses.

Complete starter code and solution sketches are withheld during blind review to avoid revealing author identity through code history, comments, or prior public versions. If chairs or reviewers require inspection, the materials can be provided through the conference's anonymized artifact-review process.

After review, the repository will be updated with complete starter code and solution sketches under an open educational license.
